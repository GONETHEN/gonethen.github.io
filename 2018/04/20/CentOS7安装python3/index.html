CentOS7安装python3

  2. 在人工智能的浪潮中沾湿衣角

安装步骤

1. 安装之前的准备工作

  • CentOS7默认安装了python2.7.5，因为一些命令比如yum要用到它

  • 执行python -V查看是否安装Python

  • 执行which python查看Python可执行文件的位置，如图可知文件在/usr/bin/目录下

  • 切换到/usr/bin/目录下，执行ll python*查看可知，python指向的是python2.7

  • 执行yum install zlib-devel bzip2-devel openssl-devel ncurses-devel sqlite-devel readline-devel tk-devel gcc make，用于下载编译python3

  • 执行mv python python.bak以备份python

####2. 安装Python3

  • 执行wget https://www.python.org/ftp/python/3.6.2/Python-3.6.2.tar.xz下载Python3，此时仍在/usr/bin目录下

  • 执行tar -xvJf Python-3.6.2.tar.xz解压Python-3.6.2.tar.xz

  • 为保险起见，先切换到/usr/local/目录下cd /usr/local/新建python3文件夹mkdir python3

  • 切换回刚刚的解压目录/usr/bin/Python-3.6.2

  • 执行./configure prefix=/usr/local/python3和make && make install进行编译安装

  • 执行ln -s /usr/local/python3/bin/python3 /usr/bin/python以创建软链接

3. 后续操作

  • yum依赖于Python2，因此需要修改yum配置。执行vi /usr/bin/yum

  • 把#! /usr/bin/python修改为#! /usr/bin/python2

  • 同理vi /usr/libexec/urlgrabber-ext-down文件里面的#! /usr/bin/python 也要修改为#! /usr/bin/python2

  • 分别执行python -V和python2 -V检查安装是否成功

  • Python3安装完成后，默认已经安装了pip，配置软链接即可使用pip工具ln -s /usr/local/python3/bin/pip3 /usr/bin/pip
